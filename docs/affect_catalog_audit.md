# 情動カタログ監査

## 結論

`conatus_engine.affect_catalog` の48件は、スピノザ『エチカ』第三部末尾の番号付き `Definitiones Affectuum` に合わせ、P3-DA-01 から P3-DA-48 までを原典の番号順に保持する。

今回の改修では、LLMが返す情動名風フィールドを廃止し、原子的特徴からPython側の48情動ルールで判定する。ただし、カタログ番号そのものは監査結果を優先し、添付仕様中の一部対応表で見られる番号ずれは採用しない。

## 重点確認

- P3-DA-17 は `Conscientiae morsus / 良心の呵責`。期待より悪く起こった過去の事物の観念を伴う悲しみとして扱う。
- P3-DA-27 は `Poenitentia / 後悔`。自由な決定によって行ったと信じる自分の行為の観念を伴う悲しみとして扱う。
- P3-DA-28 は `Superbia / 高慢`、P3-DA-29 は `Abjectio / 自己卑下`。P3-DA-27を飛ばして高慢へ詰めない。
- P3-DA-38 は `Crudelitas seu Saevitia / 残虐`。寛容は第三部末尾の番号付き48定義には独立項目として置かれていないため、現在の48件には追加しない。
- P3-DA-45 から P3-DA-48 は、過度な飲食欲、過度な飲酒欲、富への過度な欲望、肉欲に対応する。
- P3-DA-48 は `Libido / 肉欲`。添付仕様にある「その他の過度な欲望」は、この48定義の原典項目ではなく、必要ならプロジェクト独自拡張として別IDで扱うべきである。

## source_alignment

全48件に `source_alignment` を追加し、現時点ではすべて `canonical` とした。日本語名はプロジェクト内翻訳だが、番号とラテン語見出しの対応は原典番号に沿っている。

## ルール実装上の注意

- LLM出力から `gratitude`、`anger`、`remorse` などの情動名に近いフラグを受け取らない。
- P3-DA-27 後悔は、`self_action` の低い評価と `decrease_intensity` から判定する。
- 期間・長期性を要する情動は、単一Episodeでは `candidate` に留める。履歴集約で反復が確認された場合のみ `matched` へ昇格する設計とする。
